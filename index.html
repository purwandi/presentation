Latar Belakang

  1. Ujian sekarang dapat dilakukan secara online melalui peramban web
  2. Ujian saringan masuk online yang ada saat ini masih terdapat celah untuk melakukan tindak kecurangan.
  3. Dalam pelaksanaan, calon mahasiswa baru dapat membuka jendela baru di peramban web maupun perangkat lunak lainnya
  4. Perlu sebuah sistem yang dapat mengunci layar (lock screen)

Perumusan Masalah

Bagaimana merancang dan membangun sistem perangkat lunak untuk pelaksanaan ujian saringan masuk secara online?

Bagaimana mengimplementasikan lock screen pada sistem ujian saringan masuk online?

Batasan Masalah

  1. Terdiri dari beberapa topik mata pelajaran, dimana dalam sebuah topik memiliki bobot, waktu dan beberapa soal ujian
  2. Dapat berupa soal cerita, sehingga di dalam sebuah soal cerita dapat terdiri dari beberapa soal
  3. Soal ujian saringan masuk berbentuk pilihan ganda
  4. Sistem penilaian yang digunakan mempunyai rentang antara 0 sampai dengan 100
  5. Hanya bisa berjalan pada perangkat lunak web terbaru (HTML5 fullscreen API)
  6. Tidak bisa berjalan di browser handphone maupun smartphone
  7. Dalam pengembangan menggunakan PHP (Laravel Framework) dan untuk tampilan pengguna memakai Twitter Bootstrap

Tujuan

Melakukan rancang bangun sistem ujian saringan masuk online agar pelaksanaan ujian dapat di lakukan dimana saja

Mengimplementasikan fullscreen API untuk penguncian layar (lock screen) pada sistem ujian saringan masuk online

Manfaat

Mengurangi tindak kecurangan yang terjadi pada saat ujian saringan fakultas secara online dilaksanakan

Untuk pihak fakultas

  1. membantu proses administrasi
  2. menjaring calon mahasiswa baru yang memenuhi kriteria dan karakteristik masing-masing fakultas
  3. meningkatkan mutu pendidikan di fakultas

Untuk calon mahasiswa

  1. menghemat waktu dan biaya transportasi
  2. langsung mengetahui hasil ujian setelah selesai mengerjakan ujian

Metodologi Penelitian

Menggunakan metode rekayasa perangkat lunak model parallel development
