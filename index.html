Latar Belakang

1

Fakultas Ilmu Komputer ingin melakukan ujian saringan masuk calon mahasiswa baru yang dilakuka secara online

2

Ujian saringan masuk online yang ada saat ini, tidak mampu mengatasi tindak kecurangan yang dilakukan oleh calon mahasiswa baru ketika melakukan pengisian ujian online.

  1. Misalnya membuka window baru (dimana window tersebut memuat jawaban untuk menjawab soal)
  2. Soal ujian yang ditampilkan dalam sistem, masih sama antara calon mahasiswa satu dengan yang lain
3

Dengan adanya fitur fullscreen api yang terdapat di perangkat lunak peramban web modern, dimungkinkan pengerjaan ujian saringan masuk online melalui perangkat lunak peramban web dapat di kunci (lock) sehingga tidak dapat membuka window baru

Perumusan Masalah

1

Bagaimana merancang dan membangun sistem perangkat lunak untuk pelaksanaan ujian saringan masuk Fakultas Ilmu Komputer di Universitas Mercubuana Meruya secara online?

3

Bagaimana mengimplementasikan lock screen pada sistem ujian saringan masuk online?

Perumusan Masalah

  1. Ujian terdiri dari beberapa topik mata pelajaran, dimana dalam sebuah topik memiliki bobot, waktu dan beberapa soal ujian
  2. Soal ujian dapat berupa soal cerita, sehingga di dalam sebuah soal cerita dapat terdiri dari beberapa soal
  3. Soal ujian saringan masuk berbentuk pilihan ganda
  4. Sistem penilaian yang digunakan mempunyai rentang antara 0 sampai dengan 100
  5. Sistem ujian saringan fakultas hanya bisa berjalan pada perangkat lunak web terbaru, dimana perangkat lunak tersebut telah menggunakan teknologi HTML5 khususnya fullscreen API
  6. Sistem ujian saringan fakultas tidak bisa berjalan di browser handphone maupun smartphone
  7. Dalam pengembangan perangkat lunak ujian saringan masuk online menggunakan bahasa pemrogaman PHP dan memakai Laravel Framework dan untuk tampilan pengguna (user interface) penulis menggunakan Twitter Bootstrap
