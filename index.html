Latar Belakang

Ujian yang dulunya hanya dapat dilaksanakan secara konvensional sekarang dapat dilakukan secara online.

Ujian saringan masuk online yang ada saat ini masih terdapat celah untuk melakukan tindak kecurangan.

Hal ini dikarenakan calon mahasiswa baru dapat membuka jendela (window) baru di peramban (browser) web maupun perangkat lunak lainnya, dimana di dalam jendela tersebut dapat memuat jawaban untuk menjawab soal.

Perumusan Masalah

  1. Ujian terdiri dari beberapa topik mata pelajaran, dimana dalam sebuah topik memiliki bobot, waktu dan beberapa soal ujian
  2. Dapat berupa soal cerita, sehingga di dalam sebuah soal cerita dapat terdiri dari beberapa soal
  3. Soal ujian saringan masuk berbentuk pilihan ganda
  4. Sistem penilaian yang digunakan mempunyai rentang antara 0 sampai dengan 100
  5. Hanya bisa berjalan pada perangkat lunak web terbaru (HTML5 fullscreen API)
  6. Tidak bisa berjalan di browser handphone maupun smartphone
  7. Dalam pengembangan menggunakan PHP(Laravel Framework) dan untuk tampilan pengguna memakai Twitter Bootstrap

Parallel development merupakan salah satu metode pengembangan perangkat lunak yang banyak digunakan. Dengan metode parallel ini pengembang dapat membagi poses pengembangan menjadi sub-sub / cabang-cabang yang bisa di kerjakan secara parallel. Metode ini sangat efisien karena dapat mempercepat waktu pengembangan sistem perangkat lunak.
